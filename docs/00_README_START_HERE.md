## Principios del producto

- **Gratuito** — sin monetización (ver `08_FREE_PRODUCT.md`)
- **Control del usuario** — sin envío automático de CV ni auto-fill
- **Filtrado por empresa** — investigar antes de decidir aplicar
- **CV/carta bajo demanda** — en el idioma que elija el usuario
